# Embedded Systems and Microprocessors Laboratory Coursework | ITBA

This repository contains the practical assignments done for the "22.99 Microprocessors Laboratory" subject at Instituto Tecnológico de Buenos Aires. 

All assignments were developed in C for the NXP FRDM-K64F development board — built around a Kinetis K64 microcontroller (MK64FN1M0VLL12, ARM Cortex-M4F core) — using the MCUXpresso IDE.

- **TP1 - Interrupts** | Building access control system with ID and PIN entry via rotary encoder, 7 segment display and magnetic card reader

- **TP2 - Serial Communication** | UART, I²C, SPI and CAN drivers; acquisition of roll, pitch and heading from the onboard accelerometer/magnetometer over I²C; data exchange between boards over a multi-node CAN bus and visualization on PC

- **TP3 - ADC, DAC, FlexTimer, DMA, CMP** | Full-duplex Bell 202 FSK modem in two versions: ADC sampling with software demodulation, and analog comparator with timer input capture

- **TP4 - RTOS** | Access control system implemented on a real-time operating system; thread synchronization with semaphores and message queues in place of polling; logging of floor occupancy to the ThingSpeak cloud platform

- **TPF - MP3 Player** | MP3 playback from a micro SD card: FATFS, Helix MP3 decoding, DAC audio output, spectrum analyzer VU meter on a WS2812 LED matrix, graphic equalizer, and playback control
